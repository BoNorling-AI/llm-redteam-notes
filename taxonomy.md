# Red Team Taxonomy

## 1. Social Engineering
Attempts to manipulate the model emotionally or through authority framing.

## 2. Instruction Override
Attempts to override prior system instructions.

## 3. Obfuscation
Attempts to disguise harmful intent using euphemisms or encoding.

## 4. Multi-Turn Escalation
Gradually increasing the severity of a request across turns.

## 5. Prompt Injection
Malicious instructions embedded in retrieved content.
